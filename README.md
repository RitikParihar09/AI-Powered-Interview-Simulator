# AI-Powered Interview Simulator

An AI-powered interview platform that simulates technical and HR interviews
using LLMs, voice interaction, and real-time evaluation.

## Tech Stack
- React.js, Tailwind CSS
- Node.js, Express
- Firebase (Cloud Firestore)
- Local LLM + RAG
- OpenRouter (Gemini 2.0)
- Deepgram STT/TTS
